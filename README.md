# CharismaAI - Setup Instructions

## Prerequisites
- Python 3.9+
- Google Gemini API Key

## Quick Start

### 1. Setup Backend

```bash
cd backend
pip install -r requirements.txt
```

### 2. Configure API Key

Edit `backend/.env` and add your Gemini API key:
```
GEMINI_API_KEY=cd your_actual_api_key_here
```

### 3. Run Backend

```bash
cd backend
python main.py
```

Backend will run on: http://localhost:8000

### 4. Run Frontend

Open `frontend/index.html` in your browser or use a local server:

```bash
cd frontend
python -m http.server 3000
```

Frontend will be available at: http://localhost:3000

## Testing Feature 1: Agent 1 - Question Generation

1. Select a role from dropdown (or choose "Custom Role")
2. Either:
   - Upload a PDF resume, OR
   - Enter a text description about yourself
3. Click "Generate Interview Questions"
4. View personalized questions generated by Gemini
